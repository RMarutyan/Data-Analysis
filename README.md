# Data Analysis and Visualization Repository

This repository contains datasets and dashboards created using multiple tools for data analysis and visualization.

## Folder Structure

- **csv/** – Contains datasets used for analysis and visualizations.  
- **excel_dashboards/** – Dashboards created using Excel.  
- **lookerstudio/** – Dashboards created using Looker Studio.  
- **tableau/** – Dashboards created using Tableau.  

## Highlights

- Data processing and visualization performed on the included datasets.  
- Interactive dashboards created in Looker Studio and Tableau for insights and reporting.  
- Excel dashboards for quick, offline analysis.
